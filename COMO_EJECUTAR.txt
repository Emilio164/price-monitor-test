To run the application, you'll need to ensure you have Python and the necessary dependencies installed.

   1. Activate the Virtual Environment:
       * Open your terminal or command prompt.
       * Navigate to your project directory (segundo-proyecto). 
       cd "C:\Users\neo_5\OneDrive\Documentos\Otros\Desarrollo\gemini-cli\segundo-proyecto"
       * Activate the virtual environment you created earlier:
           * On Windows: .venv\Scripts\activate
           * On macOS/Linux: source .venv/bin/activate

   2. Install Dependencies (if not already done):
       * Even though we installed them during setup, it's good practice to ensure they are up-to-date. Run:
   1         pip install -r requirements.txt
       * This will install Streamlit, Playwright, BeautifulSoup, SQLAlchemy, and other required packages. Playwright browsers are already installed from our previous step.

   3. Run the Streamlit Application:
       * Once the virtual environment is active and dependencies are installed, run the following command from your project's root directory:

   1         streamlit run src/main.py

  This will launch the Streamlit application in your web browser. You can then use the sidebar to navigate between "Dashboard," "Add Product," and "Price History" sections.
